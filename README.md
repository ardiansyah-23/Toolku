# Toolku
Toolku: All-in-One File Utility &amp; OCR **Toolku** adalah aplikasi web multifungsi berbasis Streamlit untuk konversi dokumen, Excel-CSV, gambar, dan kompresi file. Dilengkapi PDF merger/splitter, batch renamer, serta OCR cerdas untuk ekstrak teks dari gambar secara instan langsung dari browser Anda dengan mudah dan cepat.
