### **Toolku: All-in-One File Utility & OCR**

**Toolku** adalah aplikasi web serbaguna berbasis Python dan Streamlit yang dirancang untuk memudahkan pengelolaan dan konversi berbagai format file secara lokal dan efisien langsung dari browser.

**Fitur Utama:**

* **Konversi Dokumen & Data:** Mengubah format Excel ke CSV, CSV ke Excel, Word ke PDF, PDF ke Word, serta ekspor laporan tabel.
* **Pengolah Gambar:** Mendukung konversi format gambar lengkap (PNG, JPG, WEBP, ICO, BMP, TIFF), konversi massal (*batch*), hingga kompresi ukuran file.
* **Manipulasi PDF:** Dilengkapi fitur penggabung (*PDF Merger*) dan pemisah halaman (*PDF Splitter*) yang praktis.
* **Pengganti Nama Massal (*Batch Renamer*):** Mengubah nama banyak file sekaligus dengan cepat dan terstruktur.
* **OCR Cerdas:** Mengekstrak teks dari gambar atau dokumen pindaian secara otomatis menggunakan teknologi *Optical Character Recognition*.
